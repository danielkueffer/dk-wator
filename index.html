Welcome to Wa-Tor

Wa-Tor is a predator-prey simulation first presented in the December 1984 issue of Scientific American in a 5-page article.

This is a Wa-Tor implementation developed in Java with various settings with which you can simulate specific population scenarios.

Read more on the Wikipedia page.

Download

  • All platforms (executable jar file): DK-Wator.jar
  • Windows (exe file): DK-Wator.exe

Manual

Just download the jar or the exe file and double click to run the simulation. There is no installation required.
